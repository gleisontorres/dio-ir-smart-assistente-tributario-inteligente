## 💡 Dicas de Ouro

### Para a Gravação:
1. **Ensaie 5 vezes** antes de gravar
2. **Cronometre** cada seção
3. **Teste a demo** pelo menos 3 vezes
4. **Grave em blocos** (problema, solução, demo, impacto) - facilita edição
5. **Tenha um backup** do demo (screenshot ou GIF)

### Para os Slides:
1. **Menos é mais** - máximo 7 palavras por slide
2. **Visual > Texto** - use imagens e ícones
3. **Contraste alto** - para legibilidade
4. **Fonte grande** - mínimo 24pt
5. **Marca d'água Bradesco** - se permitido

### Para a Apresentação:
1. **Comece forte** - estatística chocante
2. **Conte história** - pessoas se conectam com narrativas
3. **Mostre, não fale** - demo vale mais que mil palavras
4. **Termine com visão** - deixe eles querendo mais
